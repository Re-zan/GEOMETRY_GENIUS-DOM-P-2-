3. What is Event bubble ?

Event bubble is a way of handling events in DOM hierarchy. For example you have a section which contains h2, p and button. if you click the button then an event will be triggered. Now the event will "bubble up" and go to the p then it goes to the h2 and last not the least goes to the section(parent elment). That's how an event bubble works in DOM.This very important when we get a dynamic values. We can easily control the elements of a section or div. We can also easily get the data of the elements. It also makes our code shorter.
